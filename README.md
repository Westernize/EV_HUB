🚗 EV HUB – 전기차 충전소 예약 시스템

CUTE & SIMPLE EV Charging Reservation Platform

전기차 사용자는 빠르게 증가하고 있지만 대부분의 충전소는 선착순으로만 운영되고 있습니다.
기존 앱은 단순 위치 정보만 제공할 뿐, 예약·결제·쿠폰·즐겨찾기 기능이 통합된 서비스가 없습니다.
EV HUB는 이러한 문제를 해결하는 전기차 충전소 예약 플랫폼입니다.

📌 주요 기능 (Main Features)
🔐 1. 사용자 인증

회원가입(중복 검사, WELCOME 쿠폰 지급)

로그인/로그아웃(세션 기반 인증)

관리자/사용자 자동 분기

닉네임 수정

🗺️ 2. 충전소 지도 서비스

카카오맵 기반 충전소 지도

충전소 검색(이름/지역/반경)

실시간 충전소 상태 표시

마커 색상(충전 가능/충전 중/점검 중)

클러스터링 및 상세 정보 팝업

필터(사용 가능 / 예약 가능 / 완속·급속)

🔌 3. 충전소 예약 기능

날짜·시간·충전기 선택

예약 생성 / 조회 / 취소

결제 여부에 따른 상태 표시

내 예약 목록 / 사용자별 예약 조회

💳 4. 결제 시스템 (Iamport)

신용카드 결제

쿠폰 결제(WELCOME / FREE)

서버 검증 및 중복 결제 방지

결제 완료 시 NORMAL 쿠폰 지급

🎟 5. 쿠폰 시스템

쿠폰 유형: WELCOME / NORMAL / FREE

쿠폰 조회(사용/미사용)

결제 시 쿠폰 사용 가능

NORMAL 5개 → FREE 1개 교환

⭐ 6. 즐겨찾기 기능

즐겨찾기 추가/삭제

사용자별 즐겨찾기 목록

지도 및 마이페이지 연동

🛠️ 7. 관리자 페이지

회원 통계 / 예약 통계 대시보드

회원 조회·검색·수정·삭제

전체 예약 조회 및 회원별 예약 조회

🗂️ ERD (Database 구조)

User 중심 구조

User ↔ Reservation (1:N)

User ↔ Coupon (1:N)

User ↔ Favorite (1:N)

외부 충전소 데이터: CSV 기반

예약·결제·즐겨찾기 기능 중심 설계
