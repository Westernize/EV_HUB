✨ EV HUB – 전기차 충전소 예약 시스템

CUTE & SIMPLE EV Charging Reservation Platform

전기차 사용자가 급증하고 있지만 충전소는 여전히 선착순으로 운영되고 있습니다.
기존 앱은 위치 정보만 제공할 뿐, 예약·결제·쿠폰·즐겨찾기 기능이 통합된 서비스는 부재합니다.
EV HUB는 이러한 실질적인 문제를 해결하기 위한 전기차 충전소 예약 플랫폼입니다.

🔥 주요 기능
1) 사용자 인증

회원가입(중복 검사, WELCOME 쿠폰 지급)

로그인/로그아웃(세션 기반)

관리자/일반 사용자 자동 분기

닉네임 수정

2) 충전소 지도

카카오맵 기반 지도 표시

충전소 검색(이름/지역/반경)

실시간 상태(충전중/사용 가능/점검 중)

마커 색상 구분 & 클러스터링

상세 정보 오버레이

필터(사용 가능 / 예약 가능 / 완속·급속)

3) 충전소 예약

날짜·시간·충전기 선택

예약 생성/조회/취소

결제 여부(완료/미결제)에 따른 상태 표시

사용자별 예약 내역 확인

4) 결제 시스템

Iamport 결제 연동

신용카드 결제 / 쿠폰 결제

서버 검증 및 중복 결제 방지

NORMAL 쿠폰 자동 지급

5) 쿠폰 시스템

WELCOME / NORMAL / FREE

사용/미사용 구분

NORMAL 5 → FREE 1 교환

6) 즐겨찾기

즐겨찾기 추가/삭제

사용자별 즐겨찾기 목록

지도·마이페이지 연동

7) 관리자 페이지

회원/예약 통계 대시보드

회원 관리(조회·검색·수정·삭제)

전체 예약 조회 / 회원별 예약 조회

🧩 ERD 주요 구성

User 중심 구조

User ↔ Reservation (1:N)

User ↔ Coupon (1:N)

User ↔ Favorite (1:N)

충전소 데이터: CSV 기반
