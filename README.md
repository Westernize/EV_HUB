🚗 EV HUB – 전기차 충전소 예약 시스템

CUTE & SIMPLE EV Charging Reservation Platform

전기차 이용자가 급격히 증가하고 있지만, 대부분의 기존 서비스는 위치 정보 제공에 머물러 있습니다. 충전소는 여전히 선착순이며, 예약 기능이 없는 실정입니다.
EV HUB는 예약·결제·즐겨찾기 기능을 모두 통합한 국내 최초 수준의 전기차 충전소 예약 시스템입니다.

📌 1. 프로젝트 개요

전기차 증가 속도 대비 충전 인프라 부족

기존 앱은 지도와 정보 제공만 가능 (예약 기능 부재)

충전 대기·혼잡 문제 해결 필요

예약 가능한 충전소 플랫폼 개발 → EV HUB

📌 2. 주요 기능 요약
🔹 사용자 인증

회원가입(중복검사, Welcome 쿠폰 자동 지급)

로그인(세션 기반, 관리자/사용자 분기)

로그아웃, 세션 확인, 닉네임 수정

🔹 충전소 지도 서비스

카카오맵 연동 (지도 표시, 줌/드래그)

충전소 검색 (이름/지역/반경)

실시간 충전소 상태 (충전중/사용 가능/점검중)

마커 클러스터링, 상세 오버레이

필터 기능(사용 가능 / 예약 가능 / 충전 타입)

🔹 충전소 예약 기능

날짜·시간·충전기 선택 예약

내 예약 목록 조회 / 사용자별 예약 조회

예약 취소 / 예약 상태 관리(결제/미결제)

🔹 결제 시스템 (PortOne / Iamport)

쿠폰 결제(무료, 웰컴)

결제 검증 및 중복 결제 방지

NORMAL 쿠폰 자동 지급

🔹 쿠폰 시스템

WELCOME / NORMAL / FREE 쿠폰

쿠폰 조회 및 사용

🔹 즐겨찾기 기능

즐겨찾기 추가/삭제

지도/마이페이지 연동

🔹 관리자 페이지

회원/예약 통계 대시보드

회원 관리(조회·검색·수정·삭제)

예약 관리(전체 예약 / 회원별 예약 조회)
